## 📺 Verificador de Listas IPTV y Limpiador M3U

```bash
npx iptv-doctor check lista.m3u --report report.html --json report.json --csv report.csv
```

| Necesidad | Comando | Salida |
|---|---|---|
| Verificar lista | `iptv-doctor check` | HTML/JSON/CSV y badge |
| Limpiar muertos | `iptv-doctor clean` | M3U limpio |
| Corregir EPG | `iptv-doctor epg check/fix` | Auditoría JSON o M3U corregido |
| Mundial | `iptv-doctor worldcup` | XMLTV/iCal/M3U/HTML |
